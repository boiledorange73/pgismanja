Name

ST_AlphaShape — CGALアルファシェイプアルゴリズムを使って可能な凹ジオメトリを計算します。

Synopsis

geometry ST_AlphaShape(geometry geom, float alpha, boolean allow_holes = false);

説明

非常にあいまいな概念であり、解釈できる種類が多くある可能性が大きいです。アルファシェイプはそのうちの一つです。アルファシェイプは、高密度の組織化されていないデータのポイントから形状を再構築するのに使えます。事実、アルファシェイプは、元の形状の線形近似である境界線とは大きく異なります [1]。[1] F. Bernardini and C. Bajaj. Sampling and reconstructing manifolds using alpha-shapes. Technical Report CSD-TR-97-013, Dept. Comput. Sci., Purdue Univ., West Lafayette, IN, 1997. 出典: CGAL ALpha Shapes この関数は、ジオメトリ集合の凹包を計算します。が、CGALを使って、GEOSによるST_ConcaveHullとは異なるアルゴリズムを使います。参照: Concave Hulls in JTS

Availability: 3.3.0 - SFCGAL >= 1.4.1 が必要です。

This method needs SFCGAL backend.
