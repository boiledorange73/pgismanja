Name

ST_PixelAsPoint — ピクセルの左上隅のポイントジオメトリを返します。

Synopsis

geometry ST_PixelAsPoint(raster rast, integer columnx, integer rowy);

説明

ピクセルの左上隅のポイントジオメトリを返します。

Availability: 2.1.0

例

SELECT ST_AsText(ST_PixelAsPoint(rast, 1, 1)) FROM dummy_rast WHERE rid = 1;

   st_astext
----------------
 POINT(0.5 0.5)
