Name

ST_OptimalAlphaShape — 「最適な」アルファ値を計算した後にCGALアルファシェイプアルゴリズムを使って可能な凹ジオメトリを計算します。

Synopsis

geometryST_OptimalAlphaShape(geometry param_geom, boolean allow_holes = false, integer nb_components);

説明

ジオメトリ集合の「最適な」アルファシェイプを計算します。CGALは自動的に最適なアルファ値を得ることができます。このバージョンでは、得られたアルファ値から「最適な」アルファシェイプを得ます。結果は単一ポリゴンです。任意パラメータparam_allow_holesをTRUEに指定しない限りは、穴は含みません。アルファシェイプの要素数がparam_nb_components以下になるよう結果が生成されます。

Availability: 3.3.0 - SFCGAL >= 1.4.1 が必要です。

This method needs SFCGAL backend.
