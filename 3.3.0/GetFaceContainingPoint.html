Name

GetFaceContainingPoint — ポイントを含むフェイスを見つけます。

Synopsis

integer GetFaceContainingPoint(text atopology, geometry apoint);

説明

ポイントを含むフェイスのIDを返します。

ポイントがフェイスの境界に落ちた場合には例外が投げられます。

この関数は妥当なトポロジが前提であり、エッジリンクとフェイスラベルを使用して動作します。

Availability: 3.2.0
