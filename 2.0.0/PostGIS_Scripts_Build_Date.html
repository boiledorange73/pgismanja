PostGIS_Scripts_Build_Date
戻る 8.2. 管理関数 次へ

名前

PostGIS_Scripts_Build_Date — PostGISスクリプトのビルド日付を返します。

概要

text PostGIS_Scripts_Build_Date();

説明

PostGISスクリプトのビルド日付を返します。

初出: 1.0.0RC1

例

SELECT PostGIS_Scripts_Build_Date();
  postgis_scripts_build_date
-------------------------
 2007-08-18 09:09:26
(1 row)
