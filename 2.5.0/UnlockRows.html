UnlockRows
戻る 8.14. ロングトランザクション機能 次へ

名前

UnlockRows — 指定した認証IDが保持するロックを全て削除します。解放されたロックの数が返されます。

概要

integer UnlockRows(text auth_token);

説明

指定した認証IDが保持するロックを全て削除します。解放されたロックの数が返されます。

Availability: 1.1.3

例

SELECT LockRow('towns', '353', 'priscilla');
                SELECT LockRow('towns', '2', 'priscilla');
                SELECT UnLockRows('priscilla');
                UnLockRows
                ------------
                2
