ST_EndPoint
前のページ 7.4. ジオメトリアクセサ  次のページ

名前

ST_EndPoint — LINESTRINGの最後のポイントをPOINTで返します．

概要

boolean ST_EndPoint(geometry g);

説明

LINESTRINGの最後のポイントをPOINTで返します． 入力パラメータがLINESTRINGでないならNULLを返します．

このメソッドはSQL/MM仕様の実装です． SQL-MM 3: 7.1.4

この関数は三次元をサポートします．Zインデクスを削除しません．

例

postgis=# SELECT ST_AsText(ST_EndPoint('LINESTRING(1 1, 2 2, 3 3)'::geometry));
 st_astext
------------
 POINT(3 3)
(1 row)

postgis=# SELECT ST_EndPoint('POINT(1 1)'::geometry) IS NULL AS is_null;
  is_null
----------
 t
(1 row)

--3d endpoint
SELECT ST_AsEWKT(ST_EndPoint('LINESTRING(1 1 2, 1 2 3, 0 0 5)'));
  st_asewkt
--------------
 POINT(0 0 5)
(1 row)
