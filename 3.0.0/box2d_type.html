名前

box2d — 2次元の境界ボックス。ジオメトリまたはジオメトリのコレクションの2次元範囲の記述のために使われます。

説明

box2dは、ジオメトリまたはジオメトリコレクションの、2次元の囲い込んでいるボックスを表現するために使われる空間データ型です。

xmin, ymin, xmax, ymaxの値を含む表現。これらは、XとYの範囲の最小値と最大値を示しています。
