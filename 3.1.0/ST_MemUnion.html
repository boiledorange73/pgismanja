ST_MemUnion
戻る 5.13. 重ね合わせ関数 次へ

名前

ST_MemUnion — ジオメトリを結合する集約関数で、メモリを効率的に使いますが処理時間のかかるものです。

概要

geometry ST_MemUnion(geometry set geomfield);

説明

入力ジオメトリを結合し、重ね合わせ部分が無い結果ジオメトリを生成します。入力は単一ジオメトリになったり、マルチジオメトリになったり、ジオメトリコレクションになったりします。

ST_Unionと同じ結果を生成しますが、メモリ使用が少なく、処理時間が長くなります。この集約関数はジオメトリの逐次加算的結合で動作しています。これは、最初に配列を蓄積して高速アルゴリズムで結合するST_Union集約関数と反対です。

This function supports 3d and will not drop the z-index. ただし、結果はXYのみを使用して計算されます。結果のZ値は複写するか、平均値になるか、補間されます。
