ST_MinPossibleValue
戻る 6.5. ラスタバンドアクセサ 次へ

名前

ST_MinPossibleValue — このピクセルタイプが格納できる最小値を返します。

概要

integer ST_MinPossibleValue(text pixeltype);

説明

このピクセルタイプが格納できる最小値を返します。

例

SELECT ST_MinPossibleValue('16BSI');

 st_minpossiblevalue
---------------------
              -32768


SELECT ST_MinPossibleValue('8BUI');

 st_minpossiblevalue
---------------------
                   0
