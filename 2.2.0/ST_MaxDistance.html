ST_MaxDistance
戻る 8.9. 空間関係関数と空間計測関数  次へ

名前

ST_MaxDistance — ふたつのジオメトリ間の2次元距離の最大値を、投影法の単位で返します。

概要

float ST_MaxDistance(geometry g1, geometry g2);

説明

ふたつのラインストリングの最大2次元距離を投影法の単位で返します。g1とg2が同じジオメトリの場合には、ジオメトリ内の最も遠くなる頂点の距離を返します。

初出: 1.5.0

例

ポイントからラインへの基本的な最長距離

postgis=# SELECT ST_MaxDistance('POINT(0 0)'::geometry, 'LINESTRING ( 2 0, 0 2 )'::geometry);
   st_maxdistance
-----------------
 2
(1 row)

postgis=# SELECT ST_MaxDistance('POINT(0 0)'::geometry, 'LINESTRING ( 2 2, 2 2 )'::geometry);
  st_maxdistance  
------------------
 2.82842712474619
(1 row)
