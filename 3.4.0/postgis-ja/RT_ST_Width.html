Name

ST_Width — ラスタの幅をピクセル単位で返します。

Synopsis

integer ST_Width(raster rast);

説明

ラスタの幅をピクセル単位で返します。

例

SELECT ST_Width(rast) As rastwidth
FROM dummy_rast WHERE rid=1;

rastwidth
----------------
10
