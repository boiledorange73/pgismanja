TopoElement
戻る 8.9. TopoGeometryコンストラクタ 次へ

名前

TopoElement — TopoGeometryをTopoElementに変換します。

概要

topoelement TopoElement(topogeometry topo);

説明

TopoGeometryをTopoElementに変換します。

Availability: 3.4.0

例

これは完全に全て揃ったワークフローです。

-- do this if you don't have a topology setup already
-- Creates topology not allowing any tolerance
SELECT TopoElement(topo)
FROM neighborhoods;
-- using as cast
SELECT topology.TopoElementArray_Agg(topo::topoelement)
FROM neighborhoods
GROUP BY city;
