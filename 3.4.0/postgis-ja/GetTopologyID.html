Name

GetTopologyID — トポロジ名からtopology.topologyテーブル内にあるトポロジの識別番号を返します。

Synopsis

integer GetTopologyID(varchar toponame);

説明

トポロジ名からtopology.topologyテーブル内にあるトポロジの識別番号を返します。

Availability: 1.1

例

SELECT topology.GetTopologyID('ma_topo') As topo_id;
 topo_id
---------
       1
